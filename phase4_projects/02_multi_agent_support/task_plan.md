## 关键问题
1. 是否需要保留 Streamlit 还是迁移到 React？
   - 决策：保留 Streamlit，通过自定义 CSS 提升视觉效果
2. 如何优化混合检索性能？
   - 决策：添加缓存、异步处理、批量查询
3. 是否需要数据库替代文件存储？
   - 决策：使用 SQLite 存储会话和配置

## 已做决策
| 决策 | 理由 |
|------|------|
| 保留 Streamlit | 快速开发、Python 原生集成、适合 AI 应用 |
| 使用 CSS 自定义样式 | 无需重构、快速实现美观效果 |
| 添加 Redis 缓存 | 显著提升重复查询性能 |
| 使用 SQLite 持久化 | 轻量级、无需额外服务、支持会话恢复 |
